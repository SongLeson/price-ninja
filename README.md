# 🥷 Price Ninja (智选比价助手)

> **您的移动端光速比价神器**  
> "拒绝价格陷阱，像忍者一样精准出击！"

Price Ninja 是一个现代化的移动端 Web 应用，专为超市购物场景设计。它不仅能帮你迅速算出哪种包装更划算，还内置了语音交互、游戏化段位和趣味运势系统，让省钱变得好玩起来。

## ✨ 核心功能 (Key Features)

### ⚡️ 光速比价 (Smart Compare)
- **多单位换算**: 自动归一化 `克(g)`、`千克(kg)`、`磅(lb)`、`毫升(ml)` 等单位，统一计算每 100g/ml 单价。
- **智能高亮**: 自动计算**节省百分比**，并给予获胜商品 "冠军光效"。

### 🎤 忍法 · 言灵 (Voice Control)
- **语音填单**: 点击麦克风，说出 *"20块钱500克"* 或 *"3瓶10块"*，系统自动解析填入。
- **智能修正**: 自动识别并分离口语中的价格与分量，支持模糊匹配。

### 🎮 忍者修行 (Gamification)
- **段位系统**: 记录你的每一次省钱操作，积累经验值，从 **下忍** 晋升至 **影级** 忍者。
- **每日运势**: 每天在个人中心抽取 *"省钱运势签"*，查看今日宜忌。
- **金币雨**: 确认省钱后，享受解压的全屏金币奖励动画。

### 📳 体感交互 (Shake Interaction)
- **摇一摇清空**: 购物完成后，用力摇晃手机 (或点击摇一摇按钮) 即可一键重置比价数据。
- **振动反馈**: 配合真实的物理振动反馈，体验极佳。

### 📒 避雷笔记本 (Notebook)
- **红黑榜**: 记录好物与避雷商品。
- **本地存储**: 所有数据存储在浏览器 LocalStorage，无需联网，隐私安全。

## 🛠 技术栈 (Tech Stack)

- **框架**: [Vue 3](https://vuejs.org/) (Script Setup)
- **构建**: [Vite](https://vitejs.dev/)
- **状态管理**: [Pinia](https://pinia.vuejs.org/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **API**: 
  - `Web Speech API` (语音识别)
  - `DeviceMotion API` (摇一摇)
  - `Navigator.vibrate` (振动反馈)

## 🚀 快速开始 (Quick Start)

### 1. 安装依赖
```bash
npm install
```

### 2. 启动开发服务器
```bash
# 开启 --host 以支持局域网手机访问
npm run dev
```

### 3. 手机体验
确保手机与电脑连接同一 Wi-Fi，访问控制台输出的 Network 地址 (例如 `http://192.168.1.5:3000`)。

> **注意**: 部分高级功能 (如物理摇一摇权限申请) 建议在 HTTPS 环境下使用。本项目提供了 UI 兜底方案以适配 HTTP 环境。

## 📱 适配说明
本项目采用 **Mobile-First** 设计理念，针对手机竖屏操作进行了深度优化，但在桌面端也能正常浏览。
